

### Getting Started
    Clone this repository: 
        [Spring Boot Project](git@github.com:kpokala/foodtrucks.git)

### Prerequisites
    - Jdk 8+
    - Maven or maven wrapper

### Run
    mvn clean package spring-boot:run
"# foodtrucks"
